# 命令列 (CLI) 操作指南

外部連結檢查爬蟲可以完全不依賴網站後台，透過命令列直接執行與管理。請確認已進入虛擬環境，並參考以下指令操作。

## 1. 建立並啟動新爬蟲任務

爬蟲需要一個設定檔來定義起始網址 (`start_url`)、以及允許的目標網域與內部網域。
基於專案規範，建議將設定檔放置於 `jobs/` 目錄中，以免被版本控制系統追蹤。

```bash
python cli.py -c jobs/config.yaml
```

**參數說明：**
- `-c` 或 `--config`：指定個別任務的設定檔路徑。

*(註：全域設定檔 `config_global.yaml` 預設會自動讀取，用來套用超時、延遲與重試等全域限制。)*

## 2. 恢復中斷的爬蟲任務

若是爬蟲在中途中斷（例如按下 `Ctrl+C`、網路斷線等），狀態會妥善保存在資料庫中。
您可以使用 `--resume` 參數加上先前的 Job ID (UUID 格式)，讓爬蟲從未完成的佇列 (Queue) 繼續執行，而不需要重新爬取。

```bash
python cli.py --resume <JOB_ID>
```

**參數說明：**
- `--resume`：指定要恢復執行的任務 ID。

## 3. 顯示幫助說明

若忘記指令用法，可以直接執行指令（不帶引數），或是加上 `-h` 參數：

```bash
python cli.py -h
```

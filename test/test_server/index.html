Link Checker Verification Sandbox

This page contains various types of internal and external links for testing all crawler features.

  • Page 2 (internal traversal test)
  • Repeated 404 Link (Aggregation test)

2. HTTP Redirect Follow & Base URL Offset Verification

  • Redirect to Subfolder (302 Redirect test)

3. Resource Filters (Skip & Delay Rules) Verification

  • Ignored PDF File (Extension skip verification - Expect No Request, Zero Delay)

4. Error Recovery & Retries Verification

  • Temporary 503 Server Error (Expect 3 Attempts & Index Backoff Retry)
  • Slow Connection Hang (Expect Timeout Exceeded)

5. Security & Domain White-listing Verification

  • Protected Page (User-Agent header disguise verification)
  • Google (Approved Whitelisted domain)

  • Mock Social Media (Social domain HEAD downgrade test)
